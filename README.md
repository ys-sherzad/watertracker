# WATER TRACKER

**Installation**

```
# clone the repo

git clone https://github.com/ys-sherzad/watertracker.git
```

```
# on your project directory

yarn install
```

```
# install pods for iOS

cd ios && pod install
```

```
# run the app

yarn android
or 
yarn ios
```

Please create an issue if you face any problems running the project.
